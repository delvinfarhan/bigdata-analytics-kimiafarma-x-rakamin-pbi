# Big Data Analytics - Kimia Farma x Rakamin Academy

Repositori ini berisi hasil proyek dari **Program Project-Based Virtual Internship** yang diselenggarakan oleh **Rakamin Academy** bekerja sama dengan **Kimia Farma**.  
Fokus utama proyek ini adalah **Big Data Analytics**, mencakup proses *data cleaning*, *exploratory data analysis (EDA)*, serta pembuatan **dashboard interaktif menggunakan Power BI** untuk menghasilkan insight bisnis yang bernilai.

## Tujuan Proyek
- Mengimplementasikan konsep **Big Data Analytics** pada data bisnis Kimia Farma.  
- Melakukan **pembersihan dan eksplorasi data** untuk menemukan pola dan tren.  
- Membangun **dashboard interaktif** sebagai media visualisasi insight.
